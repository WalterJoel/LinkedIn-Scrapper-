# LinkedIn-Scrapper-
Un peque√±o pero retador proyecto del Bootcamp de Krowdy, netamente con fines educativos. En este proyecto se lanza un script a partir de una extension de chrome que trae informacion de todos los perfiles que coincidan con una palabra clave tal como "FullStack".

## Primero se debe instalar dependencias
```bash 
yarn
```
## Luego iniciar el proyecto
```bash 
yarn dev 
```
## Luego copiar el archivo manifest.json dentro de la carpeta dist
```bash 
ojo la carpeta dist se genera una vez lanzado el comando yarn dev
```


## Finalmente dejar ejecutando el siguiente comando para que el servidor escuche y pueda guardar datos en la BD local
```bash 
yarn dbjson
```
